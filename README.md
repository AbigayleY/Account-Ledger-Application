# Prerequisites
- Java 17
- IntelliJ IDEA
- Maven

# Account-Ledger-Application Description
This is my first big project, making an Account Ledger App.
It has a home screen that lets the user:
- Add a deposit (prompts the user for a deposit info and saves it to a csv file
- Make a (Debit) Payment- prompts the user for debit info and saves it to the csv file
- Ledger- displays the ledger screen
- Exit the application

In the Ledger screen it can: 
- show all display entries
- Display only the entries that are deposits into the account
-  Display only the negative entries (or payments)
-  A new screen that allows the user to run pre-defined reports which you can search by Month to Date, Previous Month, Year to Date, Previous Year, and Search by Vendor name
-  As well as a go back to the ledger page

-  And then a go back to Home page

  
# How to run
1. Clone the repo: https://github.com/AbigayleY/Account-Ledger-Application.git
2. Open IntelliJ IDEA
3. Click "Open" and select the project folder
4. Wait for dependencies to download 
5. Choose the "Main.java" class
6. Click the green "Run" button or press Shift + F10
